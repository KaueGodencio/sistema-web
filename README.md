# Sistema web  (CRUD)

Sistema  desenvolvido em Laravel 11, utilizando MySQL  

Como rodar o projeto

Requisitos

PHP 8.2+
Composer
MySQL (USEI O XAMPP)

Clonar o repositório

git clone https://github.com/KaueGodencio/sistema-web.git
cd sistema-web


# Instalar dependências do PHP
composer install

Configurar banco de dados
DB_DATABASE=sistema_web
 

No terminal rodar os comandos :

php artisan key:generate

php artisan migrate

php artisan db:seed

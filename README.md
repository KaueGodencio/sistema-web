# Sistema web  (CRUD)

Sistema  desenvolvido em **Laravel 11**, utilizando **MySQL** como banco de dados e **Bootstrap + CSS + JavaScript**  

Como rodar o projeto

Requisitos

PHP 8.2+
Composer
MySQL (USEI O XAMPP)

Clonar o repositório
git clone https://github.com/seu-usuario/sistema-web.git

Instalar dependências
composer install

Configurar banco de dados
DB_DATABASE=sistema_web
 

No terminal rodar os comandos 
php artisan key:generate
php artisan migrate

php artisan db:seed \\ para criar dados de testes
